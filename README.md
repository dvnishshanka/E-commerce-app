# FlexiCart E-commerce App

FlexiCart is a sample e-commerce application that allows users to view a variety of products, add desired items to their shopping cart, and manage their selections. This project demonstrates basic e-commerce functionalities, including product listing, cart management, and user authentication.

**Features**

- Product Display and Filtering:

  - Products are fetched from the Firebase database and displayed to users.
  - Users can filter products based on categories.
  - Clicking on a product displays its detailed description.

- Shopping Cart:

  - Users can add or remove products from the cart and adjust quantities as needed.
  - A summary of items in the cart is shown on the cart page.
  - Users must sign in to proceed with the checkout process.
  - Payment processing is not implemented; instead, a confirmation message is displayed after attempting to check out.

- User Authentication:

  - Users can register or sign in to the platform using Firebase Authentication.
  - User registration data is securely stored using Firebase.
